= 術語和定義 :hardbreaks: :nofooter: :icons: font :linkattrs: :imagesdir: ./media/

[role="lead"]
計費首選項控制NetApp Console如何處理您的授權和訂閱的付款、發票和續約行為。您可以在以下兩個層級之一設定偏好：

* *全球* — 適用於直接授權、市場訂閱和Keystone訂閱。
* *按訂閱計費* — 僅適用於特定市場或直接許可合約。


帳單來源可以是以下幾種：

* *市場訂閱* — 計費透過 AWS Marketplace、Azure Marketplace 或 Google Cloud Marketplace 進行。您可以選擇雲端市場提供的計費帳戶、地區和續約選項。
* *直接許可* — 計費透過NetApp。您可以在NetApp Console中管理許可證金鑰、發票聯絡人和自動續訂。
* * Keystone訂閱* — 帳單將透過您的Keystone協議進行結算。您可以在NetApp Console中管理發票收件者和成本中心標籤。


== 權限 您需要擁有 *帳戶管理員* 或 *帳單管理員* 角色才能變更帳單首選項。操作員可以查看偏好設置，但無法修改。

== 設定帳單偏好

計費偏好設定可讓您： - 在所有服務中強制使用單一發票聯絡人列表， - 將使用情況與成本中心對齊， - 管理自動續訂和到期通知， - 確保新的Cloud Volumes ONTAP (CVO) 系統繼承正確的計費帳戶。

= 帳單欄位：頁面角色：參考

== Marketplace 欄位 * *Marketplace 供應商* — AWS、Azure 或 Google Cloud。  **結算帳戶/訂閱 ID** — 用於收費的雲端結算帳戶。  **合約期限** — 期限長度和續約行為。

= 計費選項與定義：頁面角色：參考

== 條款與選項 * ** NetApp授權優先 ** — 根據有效的NetApp授權計費。更多使用權轉移到了市場。  * **僅限市場訂閱用戶** — 所有使用量均透過市場合約計費。  * **按需付費（PAYGO）** — 按實際使用量計費的彈性訂閱模式。  * **年度合約** — 固定期限的市場訂閱。  * **自動續約** — 合約到期後自動續約。  * **自訂 CVO 配置** — 進階模式，用於對應每個超大規模資料中心的多個訂閱。  * **直接許可** — 在市場容量之前使用的NetApp管理許可。  * **超大規模雲端服務供應商** — AWS、Azure 或 Google Cloud 供應商。  * **確認更改對話方塊** — 在應用程式更新之前顯示，以防止意外更改帳單。
